# KoreanSpanishAudioBot

Telegram listening-practice bot for Korean and Mexican Spanish expressions.

- Telegram: https://t.me/KoreanSpanishAudioBot
- GitHub: https://github.com/ai2bmade/KoreanSpanishAudioBot.git
- Free plan: 3 expressions per day
- Active plan: unlimited listening for manually activated users

## Local Audio Workflow

```powershell
powershell -ExecutionPolicy Bypass -File scripts\sync-audio.ps1
```
